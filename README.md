# Error Log Analyzer using Spring Boot

## Description
A full-stack log analyzer system developed using Spring Boot and MySQL.

## Features
- Log file upload
- Error detection
- Critical error analysis
- Alert system
- Time-based analysis
- Summary reports

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

## APIs
- POST /logs/upload
- GET /logs
- GET /logs/report
- GET /logs/alert
